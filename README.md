In-depth documentation on luckyswap is available at [docs.luckyswap.com](https://docs.luckyswap.com/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
